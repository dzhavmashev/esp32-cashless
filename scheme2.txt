==================== SCHEME2: ПОДРОБНОЕ МОНТАЖНОЕ РУКОВОДСТВО ====================

Назначение:
- Это гибридная схема ESP32 <-> MDB кофемашины.
- Блок UART MDB собран на быстрых оптопарах 6N137.
- Блок BILL / COIN pulse оставлен в legacy-варианте на PC817.
- Ниже все соединения даны в формате "пин -> куда подключить", чтобы собирать без двусмысленностей.


==================== 1. ГЛАВНОЕ ПРАВИЛО ПО ПИТАНИЮ ====================

Если на вашей плате ESP32 НЕТ вывода 5V:
- это не проблема
- +5V_ESP не обязано браться с пина ESP32
- +5V_ESP можно взять с отдельного внешнего блока питания 5V
- минус этого блока питания должен идти в GND_ESP32
- этот внешний 5V относится только к домену ESP32

ВАЖНО:
- 6N137 по этой схеме питается от 5V
- U1 требует +5V_ESP на pin7 и pin8
- U2 требует +5V_ISO на pin7 и pin8
- если у вас есть только 3.3V и вообще нет возможности получить 5V, то 6N137 в этой схеме использовать нельзя без переделки питания или замены компонента


==================== 2. ДВЕ ИЗОЛИРОВАННЫЕ ЗЕМЛИ ====================

В схеме есть две разные земли:
- GND_ESP32
- MACHINE_GND

ЭТИ ЗЕМЛИ НЕЛЬЗЯ СОЕДИНЯТЬ:
- не соединять проводом
- не соединять через общую "земляную" шину
- не соединять через общий минус одного и того же неразвязанного БП
- не соединять временной перемычкой для теста

Если земли соединить:
- гальваническая развязка исчезнет
- схема перестанет быть изолированной
- можно получить ложные уровни, наводки или повреждение


==================== 3. ДОМЕНЫ ПИТАНИЯ ====================

+----------------+----------------+-------------------------------------------------+--------------------------------------------------+
| Сеть           | Домен          | Для чего используется                           | Примечание                                       |
+----------------+----------------+-------------------------------------------------+--------------------------------------------------+
| +5V_ESP        | ESP32          | Питание логической стороны U1                   | Может быть от внешнего 5V БП                     |
| +3.3V_ESP      | ESP32          | Pull-up для GPIO14                              | Обычно берется с ESP32                           |
| GND_ESP32      | ESP32          | Земля ESP32, U1 pin5, U2 pin3, U3 pin2, U4 pin2 | Общая земля только ESP-домена                    |
| +5V_ISO        | MACHINE        | Питание выходной стороны U2                     | От отдельного ИЗОЛИРОВАННОГО 5V                  |
| MACHINE_GND    | MACHINE        | Земля машины, U1 pin3, U2 pin5, U3 pin3, U4 pin3| Общая земля только машинного домена              |
+----------------+----------------+-------------------------------------------------+--------------------------------------------------+

Правило по источникам питания:
- +5V_ESP и +5V_ISO не должны иметь общий минус
- +5V_ESP можно брать с обычного внешнего 5V БП в домене ESP32
- +5V_ISO должен быть отдельным изолированным 5V источником

==================== 7. БЛОК 1: U1 (6N137) — ПРИЕМ ДАННЫХ ====================
==================== MDB pin2 TXD -> GPIO14 ====================

Назначение:
- Машина передает данные по MDB pin2 TXD.
- U1 передает этот сигнал через оптическую развязку.
- На GPIO14 получается инвертированный уровень.

Логика U1:
- Idle MDB = High
- GPIO14 при Idle = Low
- MDB Low = GPIO14 High

Таблица соединений U1:

+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+
| Пин  | Имя пина  | Куда подключить                             | Через что                 | Примечание                                   |
+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+
| 1    | NC        | Не подключать                               | -                         | Оставить свободным                           |
| 2    | ANODE     | MDB pin2 TXD                                | R1 = 470 Ом               | Машинная сторона                             |
| 3    | CATHODE   | MACHINE_GND                                 | Прямое соединение         | Машинная сторона                             |
| 4    | NC        | Не подключать                               | -                         | Оставить свободным                           |
| 5    | GND       | GND_ESP32                                   | Прямое соединение         | Сторона ESP32                                |
| 6    | OUTPUT    | GPIO14 ESP32                                | Прямое соединение         | Выход U1                                     |
| 7    | ENABLE    | +5V_ESP                                     | Прямое соединение         | Обязательно                                  |
| 8    | VCC       | +5V_ESP                                     | Прямое соединение         | Обязательно                                  |
+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+

Дополнительные элементы U1:

+----------------------+-------------------------------------------+----------------------------------------------+
| Элемент              | Подключение                               | Примечание                                   |
+----------------------+-------------------------------------------+----------------------------------------------+
| R5 = 4.7 кОм         | GPIO14 -> +3.3V_ESP                       | Pull-up обязателен                           |
| C1 = 100 нФ          | U1 pin8 -> U1 pin5                        | Ставить максимально близко к U1              |
+----------------------+-------------------------------------------+----------------------------------------------+

Подробная ASCII-схема U1:

ESP32 DOMAIN                                                     ||||||  ISOLATION  ||||||   MACHINE DOMAIN

        +5V_ESP
           |
           +-------------------------------> U1 pin7 ENABLE
           |
           +-------------------------------> U1 pin8 VCC
                                              |
                                           [ C1 100nF ]
                                              |
GND_ESP32 --------------------------------> U1 pin5 GND

U1 pin6 OUTPUT ---------------------------> GPIO14
  |
  +---- R5 4.7k ----> +3.3V_ESP

MDB pin2 TXD ---- R1 470 ----> U1 pin2 ANODE
MACHINE_GND -----------------> U1 pin3 CATHODE

U1 pin1 NC -> не подключать
U1 pin4 NC -> не подключать

Краткая монтажная цепочка U1:
- MDB pin2 TXD -> R1 470 Ом -> U1 pin2
- U1 pin3 -> MACHINE_GND
- U1 pin5 -> GND_ESP32
- U1 pin6 -> GPIO14
- GPIO14 -> R5 4.7 кОм -> +3.3V_ESP
- U1 pin7 -> +5V_ESP
- U1 pin8 -> +5V_ESP
- C1 100 нФ между U1 pin8 и U1 pin5


==================== 8. БЛОК 2: U2 (6N137) — ПЕРЕДАЧА ДАННЫХ ====================
==================== GPIO19 -> MDB pin1 RXD ====================

Назначение:
- ESP32 управляет линией MDB pin1 RXD.
- U2 работает как развязанный open-collector выход на стороне машины.

Логика U2:
- GPIO19 Low -> линия MDB RXD отпущена -> Idle High
- GPIO19 High -> U2 тянет MDB RXD вниз -> Active Low

Таблица соединений U2:

+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+
| Пин  | Имя пина  | Куда подключить                             | Через что                 | Примечание                                   |
+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+
| 1    | NC        | Не подключать                               | -                         | Оставить свободным                           |
| 2    | ANODE     | GPIO19 ESP32                                | R2 = 220 Ом               | Сторона ESP32                                |
| 3    | CATHODE   | GND_ESP32                                   | Прямое соединение         | Сторона ESP32                                |
| 4    | NC        | Не подключать                               | -                         | Оставить свободным                           |
| 5    | GND       | MACHINE_GND                                 | Прямое соединение         | Машинная сторона                             |
| 6    | OUTPUT    | MDB pin1 RXD                                | Прямое соединение         | Open-collector выход                         |
| 7    | ENABLE    | +5V_ISO                                     | Прямое соединение         | Обязательно                                  |
| 8    | VCC       | +5V_ISO                                     | Прямое соединение         | Обязательно                                  |
+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+

Дополнительные элементы U2:

+----------------------+-------------------------------------------+----------------------------------------------+
| Элемент              | Подключение                               | Примечание                                   |
+----------------------+-------------------------------------------+----------------------------------------------+
| C2 = 100 нФ          | U2 pin8 -> U2 pin5                        | Ставить максимально близко к U2              |
| R6 = 4.7 кОм         | MDB pin1 RXD -> +5V_ISO                   | Только если в машине нет своей подтяжки      |
+----------------------+-------------------------------------------+----------------------------------------------+

Подробная ASCII-схема U2:

ESP32 DOMAIN                                                     ||||||  ISOLATION  ||||||   MACHINE DOMAIN

GPIO19 ---- R2 220 ----> U2 pin2 ANODE
GND_ESP32 -------------> U2 pin3 CATHODE

                                            +5V_ISO
                                               |
                                               +-----------------------------> U2 pin7 ENABLE
                                               |
                                               +-----------------------------> U2 pin8 VCC
                                                  |
                                               [ C2 100nF ]
                                                  |
MACHINE_GND -----------------------------------> U2 pin5 GND
                                                  |
MDB pin1 RXD <--------------------------------- U2 pin6 OUTPUT
      |
      +---- R6 4.7k ----> +5V_ISO    only if machine has no internal pull-up

U2 pin1 NC -> не подключать
U2 pin4 NC -> не подключать

Краткая монтажная цепочка U2:
- GPIO19 -> R2 220 Ом -> U2 pin2
- U2 pin3 -> GND_ESP32
- U2 pin5 -> MACHINE_GND
- U2 pin6 -> MDB pin1 RXD
- U2 pin7 -> +5V_ISO
- U2 pin8 -> +5V_ISO
- C2 100 нФ между U2 pin8 и U2 pin5
- при отсутствии внутренней подтяжки: MDB pin1 RXD -> R6 4.7 кОм -> +5V_ISO


==================== 9. БЛОК 3: U3 И U4 (PC817) — LEGACY PULSE ====================
==================== BILL / COIN PULSE БЕЗ ИЗМЕНЕНИЙ ====================

Pinout PC817:
- 1 = Anode
- 2 = Cathode
- 3 = Emitter
- 4 = Collector

Общее правило для U3 и U4:
- GPIO32 и GPIO33 управляют анодами через 220 Ом
- pin2 идет в GND_ESP32
- pin3 идет в MACHINE_GND
- pin4 идет на pulse input машины


-------------------- U3 (PC817) — BILL PULSE --------------------

Таблица соединений U3:

+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+
| Пин  | Имя пина  | Куда подключить                             | Через что                 | Примечание                                   |
+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+
| 1    | ANODE     | GPIO32 ESP32                                | R3 = 220 Ом               | Сторона ESP32                                |
| 2    | CATHODE   | GND_ESP32                                   | Прямое соединение         | Сторона ESP32                                |
| 3    | EMITTER   | MACHINE_GND                                 | Прямое соединение         | Машинная сторона                             |
| 4    | COLLECTOR | BILL_ACCEPTOR_PULSE_IN                      | Прямое соединение         | Выход pulse                                  |
+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+

Подробная ASCII-схема U3:

ESP32 DOMAIN                                                     ||||||  ISOLATION  ||||||   MACHINE DOMAIN

GPIO32 ---- R3 220 ----> U3 pin1 ANODE
GND_ESP32 -------------> U3 pin2 CATHODE

MACHINE_GND ---------------------------------> U3 pin3 EMITTER
BILL_ACCEPTOR_PULSE_IN <--------------------- U3 pin4 COLLECTOR

Краткая монтажная цепочка U3:
- GPIO32 -> R3 220 Ом -> U3 pin1
- U3 pin2 -> GND_ESP32
- U3 pin3 -> MACHINE_GND
- U3 pin4 -> BILL_ACCEPTOR_PULSE_IN


-------------------- U4 (PC817) — COIN PULSE --------------------

Таблица соединений U4:

+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+
| Пин  | Имя пина  | Куда подключить                             | Через что                 | Примечание                                   |
+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+
| 1    | ANODE     | GPIO33 ESP32                                | R4 = 220 Ом               | Сторона ESP32                                |
| 2    | CATHODE   | GND_ESP32                                   | Прямое соединение         | Сторона ESP32                                |
| 3    | EMITTER   | MACHINE_GND                                 | Прямое соединение         | Машинная сторона                             |
| 4    | COLLECTOR | COIN_ACCEPTOR_PULSE_IN                      | Прямое соединение         | Выход pulse                                  |
+------+-----------+---------------------------------------------+---------------------------+----------------------------------------------+

Подробная ASCII-схема U4:

ESP32 DOMAIN                                                     ||||||  ISOLATION  ||||||   MACHINE DOMAIN

GPIO33 ---- R4 220 ----> U4 pin1 ANODE
GND_ESP32 -------------> U4 pin2 CATHODE

MACHINE_GND ---------------------------------> U4 pin3 EMITTER
COIN_ACCEPTOR_PULSE_IN <--------------------- U4 pin4 COLLECTOR

Краткая монтажная цепочка U4:
- GPIO33 -> R4 220 Ом -> U4 pin1
- U4 pin2 -> GND_ESP32
- U4 pin3 -> MACHINE_GND
- U4 pin4 -> COIN_ACCEPTOR_PULSE_IN

Примечание по pulse inputs:
- pulse inputs машины должны быть Idle = High
- если внутри машины уже есть подтяжка вверх, ничего добавлять не нужно
- если подтяжки нет, она должна быть сделана на машинной стороне

==================== 11. ЛОГИЧЕСКИЕ УРОВНИ ====================

U1:
- Idle MDB (High) = GPIO14 (Low)
- MDB Low = GPIO14 High

U2:
- GPIO19 Low = MDB RXD released = Idle High
- GPIO19 High = MDB RXD pulled Low

U3:
- GPIO32 High = BILL line pulled Low
- GPIO32 Low = BILL line released

U4:
- GPIO33 High = COIN line pulled Low
- GPIO33 Low = COIN line released

==================== 13. КОРОТКАЯ СБОРКА ПО СЕТЯМ ====================

ESP32 domain:
- +5V_ESP -> U1 pin7, U1 pin8
- +3.3V_ESP -> R5 -> GPIO14
- GND_ESP32 -> U1 pin5, U2 pin3, U3 pin2, U4 pin2
- GPIO14 -> U1 pin6
- GPIO19 -> R2 -> U2 pin2
- GPIO32 -> R3 -> U3 pin1
- GPIO33 -> R4 -> U4 pin1

|||||| НЕ СОЕДИНЯТЬ ЗЕМЛИ ДОМЕНОВ ||||||

Machine domain:
- MACHINE_GND -> MDB pin3, U1 pin3, U2 pin5, U3 pin3, U4 pin3
- MDB pin2 TXD -> R1 -> U1 pin2
- MDB pin1 RXD -> U2 pin6
- BILL_ACCEPTOR_PULSE_IN -> U3 pin4
- COIN_ACCEPTOR_PULSE_IN -> U4 pin4
- +5V_ISO -> U2 pin7, U2 pin8
- при необходимости: MDB pin1 RXD -> R6 -> +5V_ISO

Развязка питания:
- C1 100 нФ между U1 pin8 и U1 pin5
- C2 100 нФ между U2 pin8 и U2 pin5
