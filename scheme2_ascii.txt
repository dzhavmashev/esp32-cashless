                      ┌─────── DC-DC ────────┐
  +VDC (~34V) ───────→│ IN+   OUT+ = +5V_ESP │──→ ESP32 VIN, U1, U2
  MDB pin6 GND ──────→│ IN−   OUT− = GND_ESP │──→ ESP32 GND, U3/U4 CA
                      └──────────────────────┘
                                ▲
                       (без доп. DC-DC: GND_ESP = MACHINE_GND — изоляции нет)
                       (для изоляции U2: нужен второй изолир. DC-DC, см. секцию 3)

  MACHINE SIDE                     ║                    ESP32 SIDE
  ────────────────────────────────  ║  ──────────────────────────────────────────
                                    ║
  MDB pin2 TXD ──[R1=470Ω]──→ U1 ──╫──→ GPIO14 ──[R5=4.7кОм]── +3V3
  MDB pin6 GND ───────────── U1 ──╫── GND_ESP32
                                    ║
  MDB pin1 RXD ←──────────── U2 ──╫──← [R2=220Ω]── GPIO19
  MDB pin6 GND ───────────── U2 ──╫── MACHINE_GND (если изол. DC-DC — своя GND)
                                    ║
  BILL_ACCEPTOR_IN ←─────── U3 ──╫──← [R3=220Ω]── GPIO32
  MACHINE_GND ───────────── U3 ──╫── GND_ESP32
                                    ║
  COIN_ACCEPTOR_IN ←─────── U4 ──╫──← [R4=220Ω]── GPIO33
  MACHINE_GND ───────────── U4 ──╫── GND_ESP32
                                    ║

 1. ПИТАНИЕ — DC-DC STEP-DOWN
══════════════════════════════════════════════════════════════════════════════════

  Машина                      DC-DC module                 ESP32-сторона
  ──────────────              ──────────────────────────    ─────────────────────

                              ┌──────────────────────┐
  MDB pin4/5 +VDC ──────────→│IN+                   │
  (~34V)                      │                      │OUT+─────────────── +5V_ESP
                              │       LM2596         │    │
  MDB pin6 GND ─────────────→│IN−    MP2307   etc.  │    ├──→ ESP32 VIN
                              │                      │    ├──→ U1 pin7 (ENABLE)
                              │                      │OUT−│    U1 pin8 (VCC)
                              └──────────────────────┘    └──→ +5V_ISO (U2, см. ниже)

                   OUT− = GND_ESP32 ────────────────────→ ESP32 GND
                                   └────────────────────→ U3 pin2 (CA)
                                   └────────────────────→ U4 pin2 (CA)
                                   └────────────────────→ U2 pin3 (CA) — если без изоляции

  ┌─────────────────────────────────────────────────────────────────────────────┐
  │  ОБ ИЗОЛЯЦИИ U2:                                                            │
  │                                                                             │
  │  Вариант A — БЕЗ изоляции (упрощённо, ваш случай):                         │
  │    DC-DC OUT− = GND_ESP32 = MACHINE_GND  (один общий минус)                │
  │    +5V_ESP = +5V_ISO (одно напряжение)                                      │
  │    Оптопары дают защиту от помех/выбросов, но не изолируют домены GND.     │
  │                                                                             │
  │  Вариант B — С изоляцией (рекомендуется):                                  │
  │    +5V_ESP (DC-DC OUT+) → [изол. DC-DC B0505S] → +5V_ISO                  │
  │                                        GND_OUT_ISO = MACHINE_GND           │
  │    MACHINE_GND и GND_ESP32 НЕ соединены.                                   │
  └─────────────────────────────────────────────────────────────────────────────┘

 2. U1 · 6N137 · MDB RX (Machine TXD → GPIO14)
    VMC передаёт → ESP32 принимает
══════════════════════════════════════════════════════════════════════════════════

  Направление сигнала: MDB pin2 TXD ──→ U1 ──→ GPIO14

  MACHINE SIDE                 U1 · 6N137                 ESP32 SIDE
  ─────────────                ───────────────────────     ──────────────────────

                               ┌───────────────────┐
  MDB pin2 TXD                 │  1(NC)   8(VCC)   │◄──── +5V_ESP
      │                        │                   │           │
      └─── [R1] ──────────────►│2(AN)     7(EN)    │◄──── +5V_ESP
           470Ω                │                   │           │
                               │      6N137         │      [C1=100нФ]
  MACHINE_GND ────────────────►│3(CA)     6(OUT)   │──────────┤ ──► GPIO14
                               │   MDB RX (U1)     │          │         │
                               │          5(GND)   │◄──── GND_ESP32   [R5=4.7кОм]
                               │  4(NC)            │                    │
                               └───────────────────┘                +3V3_ESP

  C1: 100нФ керамика, между pin8 (VCC) и pin5 (GND), вплотную к корпусу.

  Таблица уровней:
  ┌─────────────────────────────────────────────────────────────────┐
  │ MDB TXD (pin2)  │ Ток LED U1 │ U1 pin6 (OUT) │ GPIO14          │
  │─────────────────────────────────────────────────────────────────│
  │ 5V = HIGH (idle)│ ≈7мА       │ LOW           │ LOW (≈0V)   ✓   │
  │ 0V = LOW (data) │ 0мА        │ HIGH (pull-up)│ HIGH (≈3.3V) ✓  │
  └─────────────────────────────────────────────────────────────────┘
  Firmware: MDB_RX_INVERT = true  (компенсирует инверсию 6N137)

  Выбор R1 по напряжению TXD:
  ┌───────────────────────────────────────┐
  │ U_TXD   │  R1       │  IF (ток LED)  │
  │─────────────────────────────────────── │
  │  5 В    │  470 Ом   │  ≈ 7 мА   ✓    │◄── ваш случай (измерено 5V)
  │ 12 В    │  1 кОм    │  ≈ 11 мА  ✓    │
  │ 24 В    │  2.2 кОм  │  ≈ 10 мА  ✓    │
  │ 24 В    │  470 Ом   │  ≈ 48 мА  ✗    │◄── сожжёт 6N137!
  └───────────────────────────────────────┘

 3. U2 · 6N137 · MDB TX (GPIO19 → Machine RXD)
    ESP32 передаёт → VMC принимает
══════════════════════════════════════════════════════════════════════════════════

  Направление сигнала: GPIO19 ──→ U2 ──→ MDB pin1 RXD

  ESP32 SIDE                   U2 · 6N137                 MACHINE SIDE
  ─────────────                ───────────────────────     ──────────────────────

                               ┌───────────────────┐
  GPIO19                       │  1(NC)   8(VCC)   │◄──── +5V_ISO
      │                        │                   │           │
      └─── [R2=220Ω] ─────────►│2(AN)     7(EN)    │◄──── +5V_ISO
                               │                   │           │
  GND_ESP32 ──────────────────►│3(CA)     6(OUT)   │──────────┤──► MDB pin1 RXD
                               │   MDB TX (U2)     │      [C2=100нФ]
                               │          5(GND)   │◄──── MACHINE_GND
                               │  4(NC)            │
                               └───────────────────┘

  C2: 100нФ керамика, между pin8 (VCC) и pin5 (GND) — на стороне MACHINE.

  Таблица уровней:
  ┌──────────────────────────────────────────────────────────────────────┐
  │ GPIO19   │ Ток LED U2 │ U2 pin6 (OUT)           │ MDB RXD (pin1)    │
  │──────────────────────────────────────────────────────────────────────│
  │ HIGH     │ ≈14мА      │ LOW (тянет к MACH_GND)  │ LOW (данные) ✓   │
  │ LOW      │ 0мА        │ HIGH (pull-up VMC)       │ HIGH (idle)  ✓   │
  └──────────────────────────────────────────────────────────────────────┘
  Firmware: MDB_TX_INVERT = true

  Опциональный внешний pull-up (если VMC не имеет встроенного):
    +5V_ISO ──[4.7кОм]──► MDB pin1 RXD
  (Внутренний pull-up VMC обычно есть: вы измерили 48 Ом на RXD — это он.)

 4. U3 · PC817 · BILL PULSE (GPIO32 → Machine BILL input)
══════════════════════════════════════════════════════════════════════════════════

  Направление сигнала: GPIO32 ──→ U3 ──→ BILL_ACCEPTOR_PULSE_IN

  ESP32 SIDE              U3 · PC817            MACHINE SIDE
  ─────────────           ────────────────       ──────────────────────────────

  GPIO32                  ┌────────────┐
      │                   │            │
      └─── [R3=220Ω] ────►│p1(AN)  p4(COL)│──────────────► BILL_ACCEPTOR_PULSE_IN
                          │            │                        (pull-up к +VDC)
  GND_ESP32 ─────────────►│p2(CA)  p3(EMT)│◄─────────────── MACHINE_GND
                          └────────────┘

  Firmware:
    BILL_PULSE_OUTPUT_PIN        = 32
    BILL_PULSE_ON_TIME_MS        = 100
    BILL_PULSE_OFF_TIME_MS       = 100
    BILL_MINOR_UNITS_PER_PULSE   = 1000  (1 импульс = 10 рублей)

  Логика импульса:
    GPIO32 HIGH (100мс) → LED горит → p4 тянется к p3 (LOW)
    GPIO32 LOW  (100мс) → LED гаснет → p4 HIGH (pull-up машины)
    Повторить N раз для N × 1000 минорных единиц.

  ─────────────           ────────────────       ──────────────────────────────

  GPIO33                  ┌────────────┐

      └─── [R4=220Ω] ────►│p1(AN)  p4(COL)│──────────────► COIN_ACCEPTOR_PULSE_IN

 6. РАЗВОДКА ПИНОВ 6N137 (справочник)
══════════════════════════════════════════════════════════════════════════════════

       ┌──────────┐
  NC 1─┤          ├─8 VCC       ← питание выходной стороны (+5V)
  AN 2─┤  6N137   ├─7 ENABLE    ← подтяжка к VCC обязательна!
  CA 3─┤          ├─6 OUTPUT    ← open-collector
  NC 4─┤          ├─5 GND       ← земля выходной стороны
       └──────────┘

  Пины 1 и 4 — оставить незаключёнными (NC).
  Без ENABLE (pin7 = HIGH) выход OUTPUT заблокирован.

 8. СВОДНАЯ ТАБЛИЦА СОЕДИНЕНИЙ
══════════════════════════════════════════════════════════════════════════════════

  ┌──────────────────┬────────────────────────────────────────────────────────┐
  │ Откуда           │ Куда                                                   │
  ├──────────────────┼────────────────────────────────────────────────────────┤
  │ DC-DC IN+        │ MDB pin4/5  (+VDC ≈34V)                               │
  │ DC-DC IN−        │ MDB pin6    (GND машины)                               │
  │ DC-DC OUT+ (5V)  │ ESP32 VIN,  U1 p8,  U1 p7,  U2 p8*,  U2 p7*          │
  │ DC-DC OUT− (GND) │ ESP32 GND,  U1 p5,  U2 p3,  U3 p2,  U4 p2            │
  ├──────────────────┼────────────────────────────────────────────────────────┤
  │ ESP32 +3V3       │ R5 (4.7кОм) → GPIO14                                  │
  │ ESP32 GPIO14     │ U1 pin6 (OUT),  R5 pull-up к +3V3                     │
  │ ESP32 GPIO19     │ R2 (220Ом) → U2 pin2 (AN)                             │
  │ ESP32 GPIO32     │ R3 (220Ом) → U3 pin1 (AN)                             │
  │ ESP32 GPIO33     │ R4 (220Ом) → U4 pin1 (AN)                             │
  ├──────────────────┼────────────────────────────────────────────────────────┤
  │ U1 pin2 (AN)     │ R1 (470Ом) → MDB pin2 TXD                            │
  │ U1 pin3 (CA)     │ MACHINE_GND                                            │
  │ U1 pin5 (GND)    │ GND_ESP32                                              │
  │ U1 pin6 (OUT)    │ GPIO14                                                 │
  │ U1 pin7/8        │ +5V_ESP                                                │
  ├──────────────────┼────────────────────────────────────────────────────────┤
  │ U2 pin2 (AN)     │ R2 (220Ом) → GPIO19                                   │
  │ U2 pin3 (CA)     │ GND_ESP32                                              │
  │ U2 pin5 (GND)    │ MACHINE_GND*                                           │
  │ U2 pin6 (OUT)    │ MDB pin1 RXD                                           │
  │ U2 pin7/8 (VCC)  │ +5V_ISO*                                               │
  ├──────────────────┼────────────────────────────────────────────────────────┤
  │ U3 pin1 (AN)     │ R3 (220Ом) → GPIO32                                   │
  │ U3 pin2 (CA)     │ GND_ESP32                                              │
  │ U3 pin3 (EMT)    │ MACHINE_GND                                            │
  │ U3 pin4 (COL)    │ BILL_ACCEPTOR_PULSE_IN  (на машине)                   │
  ├──────────────────┼────────────────────────────────────────────────────────┤
  │ U4 pin1 (AN)     │ R4 (220Ом) → GPIO33                                   │
  │ U4 pin2 (CA)     │ GND_ESP32                                              │
  │ U4 pin3 (EMT)    │ MACHINE_GND                                            │
  │ U4 pin4 (COL)    │ COIN_ACCEPTOR_PULSE_IN  (на машине)                   │
  ├──────────────────┼────────────────────────────────────────────────────────┤
  │ MDB pin1 RXD     │ U2 pin6 (OUT)                                          │
  │ MDB pin2 TXD     │ R1 → U1 pin2 (AN)                                     │
  │ MDB pin6 GND     │ MACHINE_GND,  DC-DC IN−                                │
  └──────────────────┴────────────────────────────────────────────────────────┘

  * +5V_ISO и MACHINE_GND:
    В упрощённой схеме (DC-DC без изоляции):  +5V_ISO = +5V_ESP,  MACHINE_GND = GND_ESP32
    При использовании изол. DC-DC: +5V_ISO и MACHINE_GND — отдельный домен.

 9. ДИАГНОСТИКА GPIO14
══════════════════════════════════════════════════════════════════════════════════

  После монтажа U1, при работающем VMC:

  Мультиметр на GPIO14 (относительно GND_ESP32):

    GPIO14 ≈ 0V (LOW)    ← ПРАВИЛЬНО
      VMC запущен, TXD idle HIGH → LED U1 горит → GPIO14 LOW через транзистор

    GPIO14 ≈ 3.3V (HIGH) ← ПРОБЛЕМА → проверить:
      [ ] +5V_ESP на U1 pin7 и pin8 (без питания U1 выход заблокирован)
      [ ] MDB pin2 TXD подключён к R1 → U1 pin2
      [ ] MACHINE_GND подключён к U1 pin3
      [ ] VMC включён и работает
      [ ] R1 правильного номинала (не обрыв)

 10. CHECKLIST МОНТАЖА
══════════════════════════════════════════════════════════════════════════════════

  [ ] U2 pin7 и pin8 на +5V_ISO

  [ ] C2 (100нФ) вплотную между U2 pin8 и U2 pin5
  [ ] R5 (4.7кОм) от +3V3_ESP до GPIO14 (pull-up)
  [ ] U1 pin1 и pin4 — не подключать
  [ ] U2 pin1 и pin4 — не подключать
  [ ] Нет проводимости GND_ESP32 ↔ MACHINE_GND (если нужна изоляция)
  [ ] MDB idle: GPIO14 ≈ 0V при работающем VMC
  [ ] Pulse idle: BILL_IN и COIN_IN HIGH (pull-up машины)
